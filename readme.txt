DO GOOD 501(C) THEME
------------------------------

Thanks for using the Do Good 501(c) theme. This WordPress theme was built for non-profit organizations, but you can use it for any website.

The theme's main informational page is at:
https://www.zenwebthemes.com/shop-themes/do-good-free-501c/

------------------------------
FREE AND PREMIUM VERSIONS
------------------------------

This theme comes in two versions: Free and Premium.

Free Features

- WordPress Customizer
- Responsive design
- A social media widget
- A widgetized homepage that makes it easy to set up your site
- Call-to-Action button and headline widgets
- Large homepage photo with headlines and button

Premium Features

The Premium version has all of the free features, plus:

- Accent color
- Google Fonts
- Typography choices such as colors and sizes
- Full width, left sidebar, and right sidebar Page layouts
- Different sidebars for Posts vs Pages
- Sidebar feature widget
- Tech support
